## Lo que necesitarás

### Hardware

* Una Raspberry Pi
* Un Sense HAT (si no tienes un Sense HAT puedes recrear el proyecto en un navegador web usando el [emulador de Sense HAT](https://trinket.io/sense-hat){:target="_blank"}.)

### Software
Necesitarás la [última versión de Raspbian](https://www.raspberrypi.org/downloads/){:target="_blank"}, que ya incluye los siguientes paquetes de software:

- Python 3
- Sense HAT para Python 3

Si por alguna razón necesitas instalar un paquete manualmente, sigue estas instrucciones:

[[[rpi-install-software]]]

Escribe este comando en la terminal para instalar el paquete Sense HAT:

```bash
sudo apt-get install sense-hat
```
